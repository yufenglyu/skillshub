## 0.11.0 - 2026-07-15

Feature and maintenance release focused on the Resource Library workflow, backup portability, WebDAV migration, and a cleaner desktop UI.

### Highlights

- Add ZIP-based backups that preserve skill files as files instead of flattening downloaded skills into one JSON payload.
- Add WebDAV backup upload/import, remote backup listing, and locally persisted WebDAV connection settings. Passwords, tokens, and API keys remain excluded from backup archives.
- Add manual skill creation in the Skill Resource Library, including source/basic metadata for locally created skills.
- Add Resource Library sorting, directory view deletion, and safer folder-level delete previews.
- Move skill detail pages to a full-width layout, split notes and tags, and replace the old AI explanation tab with an AI-generated note action.
- Simplify Central Skills into a focused central-directory view and move sorting controls to the Resource Library.
- Add a top-bar theme toggle that cycles system, light, and dark modes, with refreshed VS Code-inspired light/dark themes.
- Rework the sidebar platform section with a "Software Platforms" heading and icon-only show/hide toggle.

### Improvements

- Keep GitHub-imported skill source metadata aligned with the repository name instead of generic resource-library labels.
- Group Settings directory controls in Resource Library, Central Skills, and scan directory order.
- Keep application identity and Windows MSI upgrade code stable for upgrades from older versions.
- Add a local release packaging script that produces assets matching the upstream GitHub Release naming pattern.

### Fixes

- Store app data under `~/.skillshub/db.sqlite` while preserving first-run migration from `~/.skillsmanage/db.sqlite`.
- Preserve saved WebDAV connection settings across application restarts.

## 0.11.0 - 2026-07-15 (Chinese notes)

这是一次围绕技能资源库工作流、备份迁移、WebDAV 和桌面 UI 的功能与维护版本。

### 亮点

- 备份改为 ZIP 归档，技能文件以真实文件形式保存，不再把已下载技能全部塞进一个 JSON。
- 新增 WebDAV 备份上传、导入、远端列表刷新和本机保存 WebDAV 连接配置；密码、Token 和 API Key 不会写入备份归档。
- 技能资源库新增手动创建技能，并支持为本地创建技能维护基础/来源元信息。
- 技能资源库新增排序、目录视图删除和目录级安全删除预览。
- 技能详情页改为全宽布局，备注与标签拆分，并把原 AI 解释标签页调整为备注区的 AI 生成备注按钮。
- 中央技能库聚焦中央目录管理，排序能力移到技能资源库。
- 主界面右上角新增主题切换按钮，可在系统、浅色、深色之间轮询，并刷新为参考 VS Code 的浅色/深色主题。
- 左侧栏软件平台区新增标题和图标，仅保留显示/隐藏空平台的图标按钮。

### 改进

- GitHub 导入技能的来源信息保存为仓库名，避免显示为泛化的 resource-library。
- 设置页目录相关配置按技能资源库、中央技能库、扫描目录顺序排列。
- 保持应用身份与 Windows MSI upgrade code 稳定，支持从旧版本升级。
- 增加本地 release 打包脚本，产物命名与 GitHub Release 约定保持一致。

### 修复

- 应用数据保存到 `~/.skillshub/db.sqlite`，并保留从 `~/.skillsmanage/db.sqlite` 首次迁移的兼容逻辑。
- 修复 WebDAV 登录/连接配置重启后清空的问题。
