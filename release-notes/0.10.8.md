## 0.10.8 - 2026-07-14

Feature release focused on marketplace bulk installation, resource library cleanup, and automated desktop release packaging.

### Improvements

- Add an "Install all" action for expanded marketplace repository previews.
- Add deletion support for installed resource library skills, including confirmation for skills linked to agent platforms.
- Align desktop release asset names with the current `skillshub` binary and product naming.
- Build macOS release assets as universal `.dmg`, `.zip`, and `.tar.gz` packages in GitHub Actions.

### Fixes

- Prevent resource library deletion from removing files outside the configured resource root.
- Cascade platform uninstall records when deleting a resource library skill with linked platform installs.
- Fail local cross-platform packaging early with a clear message instead of surfacing Tauri host-specific bundle errors.
- Preserve Linux AppStream release metadata formatting when bumping versions.

## 0.10.8 - 2026-07-14（中文）

本次版本聚焦技能市场批量安装、资源库删除能力，以及桌面端自动发布打包流程。

### 改进

- 技能市场仓库预览新增“一键安装全部”。
- 技能资源库已安装技能支持删除；如果技能已安装到平台，会先确认并支持级联卸载。
- 桌面端发布产物统一使用当前 `skillshub` 命名。
- GitHub Actions 自动构建 macOS universal `.dmg`、`.zip` 和 `.tar.gz` 产物。

### 修复

- 删除资源库技能时限制路径范围，避免误删资源库外文件。
- 删除资源库技能时同步清理平台安装记录。
- 本地跨平台打包会提前给出明确错误，不再落到 Tauri 的主机平台限制错误。
- 修复版本更新时 Linux AppStream release 元数据格式异常。
