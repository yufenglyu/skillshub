## 0.10.7 - 2026-07-14

Maintenance release focused on moving SkillsHub configuration data to the new product-specific directory while preserving upgrade compatibility.

### Improvements

- Use `~/.skillshub/db.sqlite` as the default SQLite database path.
- Keep application identity and Windows MSI upgrade code stable for upgrades from older versions.

### Fixes

- Store app data under `~/.skillshub/db.sqlite` while preserving first-run migration from `~/.skillsmanage/db.sqlite`.

---

## 0.10.7 - 2026-07-14 (中文说明)

本版本将 SkillsHub 的配置数据迁移到新的产品专属目录，同时保留旧版本升级兼容性。

### 改进

- 默认 SQLite 数据库路径改为 `~/.skillshub/db.sqlite`。
- 继续保持应用 identifier 和 Windows MSI upgradeCode 稳定，保证旧版本安装包可以升级。

### 修复

- 首次启动时，如果新数据库不存在，会从旧的 `~/.skillsmanage/db.sqlite` 自动复制迁移。
