# 更新日志

本文件记录该项目的重要变更。

## 0.12.0 - 2026-07-20

本版本聚焦软件平台管理、项目发现行为和分平台打包脚本。

### 新功能

- 将设置页中的扫描目录和自定义平台整合为更清晰的平台与项目目录管理区域。
- 将管理对象拆分为软件平台和项目目录，并支持查看内置软件平台。
- 新增 Windows、macOS、Linux 三个独立发布打包脚本，每个脚本只负责自己的宿主平台。

### 改进

- 本地不存在对应 skills 目录时，主界面不再显示该内置软件平台。
- 优化设置页布局，把新增操作放到对应分区标题旁边。
- 更新中英文设置页截图以匹配当前界面。
- 优化 README 中的打包说明，改为平台专用命令和统一参数表。
- 停止追踪 `docs/` 下的本地规划文档，并从仓库移除生成型 `release-notes` 文件。

### 修复

- 未配置项目目录时，浏览器 fixture 中的项目技能不再出现在项目技能库。
- 修复 Homebrew Rust 覆盖 rustup toolchain 时 macOS universal 打包找不到目标标准库的问题。
- 修复 macOS 和 Linux 打包脚本中 `mktemp` 模板的跨平台兼容性。
